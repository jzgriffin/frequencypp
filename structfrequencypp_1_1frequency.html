frequencypp::frequency struct

Represents a temporal frequency.

A frequency consists of a count of ticks of type Rep and a tick period Period, where the tick period is a compile-time rational fraction representing the frequency in hertz from one tick to the next.

The only data stored in a frequencypp::frequency is a tick count of type Rep. If Rep is floating point, then the frequencypp::frequency can represent fractions of ticks. Period is included as part of the type and is only used when converting between different frequencies.
