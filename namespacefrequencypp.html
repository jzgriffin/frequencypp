template<typename ToFrequency, typename Rep, typename Period>
auto frequencypp::frequency_cast(const frequency<Rep, Period>& f) -> std::enable_if_t< detail::is_frequency_v< ToFrequency >, ToFrequency > constexpr

Convert a frequency::frequency to a frequency of different type ToFrequency.

Template parameters
ToFrequency frequency::frequency type to convert to
Rep arithmetic type representing the number of ticks for f
Period ratio representing the tick period for f
Parameters
f frequency to convert
Returns f converted to a frequency of type ToFrequency

No implicit conversions are used. Computations are done in the widest type available and converted, as if by static_cast, to the result type only when finished.
